## TC001 – Register User

**Feature:** Authentication

**Steps**
1. Navigate to https://automationexercise.com
2. Click "Signup / Login"
3. Enter name and email
4. Click "Signup"
5. Fill registration form
6. Click "Create Account"

**Expected Result**
User account should be successfully created.
